Msekeli Mkwibiso

Front-end developer, Web designer, & Graphic designer

I am a self-taught programmer with knowledge in both front-end and back-end development. My passion for programming dates back to 2018. I was introduced to Java
(a back-end programming language) through a training program offered by Capaciti.

After finishing the Java training, I started to learn front-end development using a site w3schools starting with the mark-up languages HTML5 and CSS to create websites. Upon realizing that I am more passionate about front-end development, I then decided to improve my user interface skills by learning JavaScript and the front-end framework Bootstrap.

In 2021 I received a bursary to study a short course in graphic design at the Academy of digital arts where I learned how to use design tools (Photoshop and Illustrator), which I use when designing and building websites.
